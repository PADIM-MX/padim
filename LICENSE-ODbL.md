# Open Database License (ODbL) v1.0 — Datos

## Aplicación

Los datos inmobiliarios distribuidos a través de PADIM (el "Contenido de la Base 
de Datos") están bajo la **Open Database License (ODbL) v1.0** de Open Data Commons.

## ¿Qué implica ODbL?

| Derecho | Permiso |
|---------|---------|
| Compartir | ✅ Copiar, distribuir y usar la base de datos |
| Crear | ✅ Producir obras derivadas de la base de datos |
| Adaptar | ✅ Modificar, transformar y construir sobre la base de datos |
| **Atribución** | ⚠️ Debes dar crédito a PADIM como fuente |
| **Share-Alike** | ⚠️ Si compartes una versión modificada/mejorada, debe estar bajo ODbL |
| **Mantener abierto** | ⚠️ No puedes restringir el acceso a los datos derivados |

## Excepción Comercial

BACKBONE (`https://back-bone.dev`) tiene una **licencia comercial separada** que
le permite usar los datos ODbL como parte de su API SaaS sin requerir que sus
productos derivados se compartan bajo ODbL. Contacta a info@back-bone.dev para
licencias similares.
