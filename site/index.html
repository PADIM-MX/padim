📦 CLI — pip install padim

Scrapea, valida y contribuye datos desde tu terminal. Sin servidores centrales. Sin registros. Sin permiso.

bash ~$
# Instalar
pip install padim

# Scrapear una colonia
padim scrape vivanuncios --colony "Del Valle" --output delvalle.json

# Validar contra el schema PADIM
padim validate delvalle.json

# Calcular trust scores
padim quality delvalle.json

# Servir API local
padim serve --port 8080
