CLI

pip install padim

Scrapea, valida y contribuye datos desde tu terminal. Sin servidores centrales. Sin registros. Sin permiso.

bash
# Instalar
pip install padim

# Instalar un scraper (descarga del repositorio comunitario)
padim scrapers install vivanuncios

# Scrapear una colonia
padim scrape vivanuncios --colony "Del Valle" --output delvalle.json

# Validar contra el schema PADIM
padim validate delvalle.json

# Calcular trust scores
padim quality delvalle.json

Open Source

Índice ligero, datos en la red

PADIM no almacena datos completos. PADIM es el índice. Como OpenStreetMap almacena coordenadas y metadatos —no la foto satelital—, PADIM guarda fingerprints y metadatos mínimos. Los datos completos se distribuyen a través de GitHub Releases e IPFS.
